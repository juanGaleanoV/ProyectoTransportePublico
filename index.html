Bienvenido a Mi Proyecto de Transporte Público a continuación les expongo mis historias de usuario:

1. HU1 Portal WEB: Como desarrollador quiero poder entregar un portal WEB para una empresa de transporte público que permita generar consultas a los usuarios.

2. HU2 Login de Usuario: Como usuario quiero poder registrarme y loguearme en el portal para así crear mi historia en la aplicación y poder generar consultas de ruta más usada y demás información.

3. HU3 Pago On Line: Como usuario quiero poder realizar pagos ON Line o Recargas para no tener que realizar pagos en efectivo.

4. HU4 Consulta de información: Como usuario quiero poder validar el estado de la ruta, cantidad de asientos disponibles, información del conductor, etc.

5. HU5 Calidad del Servicio: Como dueño encuestas para medir la satisfacción de los usuarios y poder tomar decisiones de manera proactiva.

6. HU6 Integraciones: Como dueño de la aplicación deseo poder conectarme a las bases de datos gubernamentales publicas sobre tráfico, cambios viales, TRM.
